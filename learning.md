## Working with .tar files 

### Checking content
```console
tar -tf filename.tar.gz
```

### Extracting the content 
```console
tar -xf archive.tar
tar -xvf archive.tar # showing the name of the file extracted 
tar -xf archive.tar file1 file2 # extracting files
```
